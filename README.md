# Playwright E2E Automation Framework

Proyecto final de automatización de pruebas de interfaz de usuario (UI) utilizando **Playwright** y **TypeScript** sobre la plataforma demo [Practice Software Testing](https://practicesoftwaretesting.com/).

## Características
- Patrón de diseño Page Object Model (POM).
- Pruebas automatizadas en múltiples navegadores (Chromium, Firefox, WebKit).
- Integración Continua con GitHub Actions.
